# Industrial-Generator-Data-Acquisition-from-DG-Controller-via-RS485-Modbus-RTU-
Overview
This project demonstrates real-time data acquisition from a Kirloskar KG645 Generator Controller using RS485 Modbus RTU communication. An ESP8266 (NodeMCU) acts as a Modbus master, reads operational parameters from the controller, and publishes them to an MQTT broker for remote monitoring.
The project was developed to evaluate the feasibility of extracting generator data through the controller's communication interface and integrating it with IoT-based monitoring systems.

Features:
Real-time generator data acquisition
RS485 Modbus RTU communication
ESP8266-based data gateway
MQTT data publishing
Live monitoring using MQTTBox
Industrial IoT architecture
Scalable for cloud integration
