# Industrial-Generator-Data-Acquisition-from-DG-Controller-via-RS485-Modbus-RTU-
Developed an IoT-based generator monitoring system using ESP8266, RS485, and Modbus RTU to acquire real-time parameters from a Kirloskar KG745 DG controller. Published data to an MQTT broker and monitored engine status, fuel level, RPM, temperature, and power through MQTTBox.
